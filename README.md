# How Accurate Are Survey Forecasts on the Market?

This repository contains the data used in:

**"How Accurate Are Survey Forecasts on the Market?"**
by Songrun He, Jiaen Li, Linying Lv, and Guofu Zhou

Paper: [https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4390165](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=4390165)

## Data

The `Data/` folder contains all input data files used in the analysis, including survey data, market returns, sentiment indices, uncertainty measures, and macro predictors.
